# Willis-Dynasty-Trust-8

Here's your step-by-step to pull and run it locally:

1 — Clone the repo

```bash
git clone https://github.com/teaw153-stack/Willis-Dynasty-Trust-8
cd Willis-Dynasty-Trust-8
```

2 — Install dependencies

```bash
npm install
```

3 — Start the app

```bash
npx expo start
```

Expo opens a QR code in your terminal. Scan it with Expo Go (free on the App Store) and you're live.

Already have it cloned? Just pull latest + run:

```bash
cd Willis-Dynasty-Trust-8
git pull origin main
npm install
npx expo start
```

Two things you need:

* Expo Go installed on your iPhone (App Store, free)
* Your phone + Mac on the same WiFi network

Backend is already live — no need to run anything locally for that. constants/config.js is already pointed at:

https://bossyboo-5e1a.onrender.com

So the full flow — signup → dashboard → upgrade → billing portal — works from the moment you scan the QR code. 🚀
